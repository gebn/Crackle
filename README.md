# Crackle

Crackle is a powerful yet easy to use HTTP client. It supports:

 - GET, POST
 - Authentication (basic, digest, NTLM)
 - File uploads
 - Simultaneous processing of multiple requests
 - Several others...

## Requirements

Crackle requires PHP >= 5.5.0 with the cURL extension installed.
